Lisk-js is a library to use Lisk cryptography, API and nodes which can be used with NodeJS or in the Browser.

THe library has been crafted with the developer in mind. Easily use the functions to connect to the Lisk API.

Lisk uses brainwallets for the user authentification. This implies that every user has access to a secret which identifies with her/his account.

When using lisk-js the functions you are using get sure that your request will get submitted to a trusted lisk node.

Every transaction using a secret will be cryptographically wrapped inside the functions so that just finished transactions get submitted to a node.

No secret leaves your browser using lisk-js.
